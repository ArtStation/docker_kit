# KuberKit

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'kuber_kit'
```

## Development

### Launch compilation

```
bin/kit compile ruby_app,ruby_app2 --path=./example
```

### Launch deployment

```
bin/kit compile -s ruby_app,ruby_app2 --path=./example
```
